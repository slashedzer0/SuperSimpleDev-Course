### Explanation 

```markdown
  - <p>...</p> is an element
  - <button> is an opening tag
  - </button> is a closing tag
  - <a>...</a> is an element to link to another website
  - href="..." is an attribute
  - href is the attribute name
  - "..." is the attribute value
  - you can put more than one attribute in an element
  - extra spaces are ignored
  - extra new lines are ignored
  ```
