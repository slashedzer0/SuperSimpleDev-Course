### Explanation

```markdown
  - create element first then style it one-by-one
  - <style>...</style> is unique element
  - it is used to style the page (to write CSS)
  - it is not physically displayed on the page like other elements
  - .subscribe-button, .join-button, .tweet-button are CSS selectors
  - period (.) in front of the class name is used to select which element we're targeting
  - background-color, color, height, width, etc. are CSS properties
  - #cc0000, #fff, none, 15px, pointer, etc. are CSS values
```
